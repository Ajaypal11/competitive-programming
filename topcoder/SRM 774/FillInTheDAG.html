Single Round Match 774 1000 FillInTheDAG

Problem Statement

You are given a directed acyclic graph with n nodes and m edges. The edges are given in the int[]s f, t denoting a directed edge from node f[i] to node t[i]. The nodes are given in topological order (so f[i] < t[i] for all i). Node 0 is guaranteed to be the only node with no incoming edges. Node n-1 is guaranteed to be the only node with no outgoing edges.

You want to label nodes with rational numbers such that the following holds.

Let x_v be the value of the v-th node. The node 0 must have value x_0 = 0 and the node n-1 must have value x_{n-1} = 1. All other nodes must satisfy 0 ≤ x_v ≤ 1.

For all non-source and non-sink nodes, x_v must be the average of the smallest value of a node that it can reach and the largest value of a node that can reach it. More formally, let S(v) be the set of nodes reachable from v (not including v), and let T(v) be the set of nodes that can reach v (not including v). Then, x_v must be equal to ((min_(y in S(v)) x_y) + max_(z in T(v)) x_z)) / 2 for all nodes v from 1 to n-2.

Return this as a long[] of nonnegative integers with the following format { p_0,q_0, p_1,q_1, ..., p_{n-1},q_{n-1} }. Here, this represents setting x_i = p_i/q_i (p_i and q_i do not need to be coprime). If there are multiple possible answers, output any of them. If there are no answers, return an empty long[]. It is guaranteed under the current constraints that if there is an answer, then there is one where all results fit in a signed 64-bit integer.

Limits

  • Time limit (s) 4.000
  • Memory limit (MB) 256

Constraints

  • n will be between 3 and 100.
  • m will be between 2 and 1,000.
  • f,t will contain exactly m elements each.
  • 0 <= f[i] < t[i] < n for all valid i.
  • Node 0 will be the only node with indegree zero.
  • Node n-1 will be the only node with outdegree zero.
